# Archived Docs

This directory holds dated kickoff plans, baseline snapshots, and historical
status notes. These files are kept for audit history and context, but they are
not the current execution plan.

Current handoff docs live one level up in [../README.md](../README.md).

## Contents

| Path | Meaning |
| --- | --- |
| [planning/](planning/) | Early migration, dashboard, and scheduler V2 planning notes. |
| [baseline/](baseline/) | Initial CarbonEngine checkout and harness status snapshots. |

When a current review doc references an archived file, treat that reference as
historical evidence context. Use `target/carbon/evidence/` for current measured
status.
